Macro CPPAN_DEFINE_MEMBER_DETECTOR

CPPAN_DEFINE_MEMBER_DETECTOR

Synopsis

// In header: <cppan/cppan.hpp>

CPPAN_DEFINE_MEMBER_DETECTOR(X)

Description

Stolen from http://en.wikibooks.org/wiki/More_C%2B%2B_Idioms/Member_Detector Define metafunction that detect presence of some member in structure. Has specialization for cppan::annotated_member< MemberType, AnnotationsType >, which step into AnnotationsType to check annotation presence.
