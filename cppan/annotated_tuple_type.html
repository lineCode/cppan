Struct template annotated_tuple_type

cppan::annotated_tuple_type

Synopsis

// In header: <cppan/cppan.hpp>

template<typename T> 
struct annotated_tuple_type {
  // types
  typedef T::annotated_tuple_type type;
};

Description

Depending on constness of type T, return either T::annotated_tuple_type or T::const_annotated_tuple_type. This can be helpfull for those who don`t have C++11 auto and decltype keywords.
