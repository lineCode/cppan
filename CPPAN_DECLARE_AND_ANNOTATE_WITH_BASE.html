Macro CPPAN_DECLARE_AND_ANNOTATE_WITH_BASE

CPPAN_DECLARE_AND_ANNOTATE_WITH_BASE

Synopsis

// In header: <cppan/declare_and_annotate.hpp>

CPPAN_DECLARE_AND_ANNOTATE_WITH_BASE(BasesSeq, MembersTuple)

Description

Declare and annotate members with specified types. Accept Boost.Preprocessor sequence of member field declaration. Each element in sequence must be Boost.Preprocessor tuple with 3 elements (member_type, member_name, sequence_of_annotations). 

sequence_of_annotations := sequence_of_annotations | CPPAN_NIL_SEQ

CPPAN_NIL_SEQ can be used if member doesn`t have annotations. sequence_of_annotations - is Boost.Preprocessor sequence of tuples with 2 elements (annotation_name, annotation_runtime_value). 

struct D : B1, B2
{
    CPPAN_DECLARE_AND_ANNOTATE_WITH_BASE(
        (B1)(B2),
        ((int, int_field_,
            ((int_annotation, 24))
            ((string_annotation, "Privet"))
        ))
        ((std::string, string_field_,
            ((no_serialization, std::true_type()))
            ((no_hash, std::true_type()))
        ))
        ((double, no_ann_field_, CPPAN_NIL_SEQ))
      )
};
